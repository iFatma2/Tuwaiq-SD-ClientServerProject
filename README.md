## Description
This is a small experimental project developed during **Software Development** bootcamp at Tuwaiq Academy.  
The goal is to try sending messages and data between devices using different programming languages and methods.

The project has **three components**:

---

## 1️⃣ Python Messenger (Flask + Tkinter)

**Description:**  
A desktop application that allows sending messages from one device (Mac) to another device (iPhone or browser).  
Messages are stored in memory and displayed on a simple web page.

**Features:**
- Tkinter GUI to type messages
- Flask server to display messages
- Real-time message updates in browser

**Technologies:** Python, Tkinter, Flask, Threading

**File:** `chat.py`

---

## 2️⃣ Python Email Sender (Tkinter)

**Description:**  
A desktop application to send emails using Gmail SMTP.  
Supports writing subject, body, and attaching files.

**Features:**
- Tkinter GUI for input
- Send emails via Gmail SMTP
- Attach files to emails

**Technologies:** Python, Tkinter, SMTP, EmailMessage

**File:** `EmailSender.py`

---

## 3️⃣ SwiftUI iOS App

**Description:**  
An iOS app built with SwiftUI to send messages from iPhone via SMS, Email, or AirDrop.

**Features:**
- Send SMS using native iOS message composer
- Send Email using native iOS mail composer
- Share messages via AirDrop
- Clean SwiftUI interface with buttons

**Technologies:** SwiftUI, MessageUI, UIActivityViewController

**File:** `ContentView.swift`
